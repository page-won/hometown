Settings

Name Type Description
Initialization -- Add the class scroll-box to the component and the attribute data-height="value" to set the height in px of the box when it's closed. This attribute accept various values, see below for more details.
Height Attribute
data-height="123" Fixed height in px.
data-height=".your-class" Use the height of the parent container with class .your-class.
data-height="#your-id" Use the height of the parent container with ID your-id.
data-height="fullscreen" Use the height of the user's screen.
Add the attribute data-offset="123" to adjust the height when it's setted to ID, class or fullscreen, this feature subtract the offset value from the original height.
Colors attribute Add the attribute data-rail-color="#c3dff7" to change the rail color and the attribute data-bar-color="#379cf4" to change the background bar color. To get the color you want visit htmlcolorcodes.com/color-picker/
Responsive CSS Add the class disable-md to the component to disable the scroll only on medium screens and the class disable-sm to disable the scroll only on small screens.
Options Attribute You can pass the plugin options directly from the component with the attribute data-options="optione:value,optione:value, ...". To view the full options list visit the plugin documentation page.
Scroll position Information The component automatically add the class scroll-pos-top when the scroll reach the top and the class scroll-pos-bottom when it reach the bottom. The class scrolling is added while scrolling.
