Settings

Name Type Description
Animations Attribute Add the attribute data-tab-anima="animation-name" to the component to apply an animation when the tab change. For the animations list and for more details visit the animations page.
Icon tab -- Add the class tab-icon to the component and insert into each nav item the code <i class="icon-name"></i>. For the icons list and for more details visit the icons page.
Inverted tab -- Add the class tab-inverse to the component and move the navigation block on bottom, after the panels.
Menu alignment -- Add the class align-right or align-center to the <tab-nav> ... </tab-nav> menu to align it on right or on center.
