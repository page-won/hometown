Usage

<a href="image.jpg" data-lightbox-anima="fade-in" class="btn btn-sm lightbox" data-options="mainClass:inner" title="Lorem ipsum">Image</a>

<a href="https://www.youtube.com/watch?v=Lb4IcGF5iTQ" data-lightbox-anima="fade-in" class="btn btn-sm lightbox">Youtube</a>

<a href="https://vimeo.com/162564288" data-lightbox-anima="fade-in" class="btn btn-sm lightbox">Vimeo</a>

<a href="video.mp4" data-lightbox-anima="fade-in" class="btn btn-sm lightbox">MP4</a>

<a href="https://maps.google.com/maps?q=221B+Baker+Street,+London,+United+Kingdom&hl=en&t=v&hnear=221B+Baker+St,+London+NW1+6XE,+United+Kingdom" data-lightbox-anima="fade-in" class="btn btn-sm lightbox">Map</a>

<a href="#your-id" class="btn btn-sm lightbox" data-lightbox-anima="fade-in">Custom</a>
<div id="your-id" class="box-lightbox">
    <div class="scroll-box" data-height="400">
        Lorem ipsum dolor sit amet, consectetur adipisicing.
    </div>
</div>

Settings

Name Type Description
Initialization CSS Add the class lightbox to a link the enable the lightbox.
Javascript functions Javascript
Lightbox enabled Use the function $('#target').magnificPopup({ type: 'content-type' }); to make a link lightbox.
Open a lightbox Use the function $.magnificPopup.open({ items: { src: 'content-link' }, type: 'content-type' }); to open a lightbox directly from Javascript.
Animations Attribute Add the attribute data-lightbox-anima="animation-name" to the link show an animation when the user open the lightbox. For the animations list and for more details visit the animations page.
Show on load HTML To show a lightbox content on page load use the following code: 
<div id="your-id" class="box-lightbox" data-lightbox-anima="show-scale" data-trigger="load">
    Lorem ipsum dolor sit amet, consectetur adipisicing.
</div>
Below the available options:
data-link="URL" Show a media content like a video. Insert e Youtube or Vimeo link.
data-click="URL" Redirect to a new page when the user click the box.
data-click-target="_blank" Open the link in a new window.
data-expire="123" Show the lightbox only the first time and every 123 days. Set to 0 to reset.
Show on scroll HTML To show a lightbox content when the user scroll the page and reach the desidered area use the following code: 
<a href="your-id" class="lightbox-trigger" data-trigger="scroll"></a>
<div id="your-id" class="box-lightbox" data-lightbox-anima="show-scale" data-trigger="scroll">
    Lorem ipsum dolor sit amet, consectetur adipisicing.
</div>
The link with the class lightbox-trigger must be placed at the position of the page that will activate the lightbox, it can be hidden but must not use CSS value display:none.
Youtube and video attributes String If you need to add some attribute to Youtube (or other) link. Use ? instead of &. Example: https://www.youtube.com/watch?v=HIfDatvJbLM?rel=0 rel=0 is the attribute.
Global lightbox CSS Add the class lightbox-global to any container to convert every image link inside it to lightbox.
Deep linking URL
Grid list For more details visit the grid list page.
Masonry list For more details visit the masonry list page.
Single item Open a lightbox item with the URL www.your-site.com?lightbox=YOUR-ID where YOUR-ID is the ID of the lightbox item you want to open. Supported components: image-box, buttons, custom lightbox with class box-lightbox and all the components with class lightbox.
Try it Open item 1 Open item 2
Options Attribute You can pass the plugin options directly from the component with the attribute data-options="optione:value,optione:value, ...". To view the full options list visit the plugin documentation page.
Lightbox types URL
Image URL of the image. To show a text caption add the attribute title="Your text" to the lightbox link.
Video MP4 URL of the MP4 video. The video must be compatible with the HTML5 format. You can use the free software Any video converter to convert your videos.
Video Youtube URL of the URL of the Youtube video. Only the format https://www.youtube.com/watch?v=VIDEO-ID is supported.
Video Vimeo URL of the URL of the Vimeo video.
Google Maps URL of the Google Map with the following format: https://maps.google.com/maps?q=YOUR-ADDRESS. YOUR-ADDRESS must be replaced with an address, replace the spaces of the address with the + symbol. You can copy the address from the URL of your browser. Ex: the URL to use for the map of the following link is https://maps.google.com/maps?q=New+York,+United+States. Link.
Custom HTML To show a custom HTML content set the href attribute of the lightbox link to the ID or class of custom HTML item and add the class box-lightbox. You can set different sizes for the box by add one of the following classes to the custom HTML item: box-lightbox-sm,box-lightbox-md,box-lightbox-lg,full-screen.
Try it Small Medium Large Full screen
Scroll box To show a scrollable box insert the content into the component <div class="scroll-box" data-height="123"> ... </div>. The scroll box mode require the following script installation: 
<script src='themekit/scripts/slimscroll.min.js'></script>
For more details visit the scroll box page.
Lightbox gallery Information Visit the grid list or the masonry list page for more informations.
