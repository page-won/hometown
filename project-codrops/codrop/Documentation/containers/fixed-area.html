Settings

Name Type Description
Top offset Attribute Add the attribute data-offset="123" to the component to set the initial sticking position in pixels.
Parent Attribute Add the attribute data-parent="[CSS-SELECTOR]" to the component to maintian the fixed area correctly positionation on scroll. Replace [CSS-SELECTOR] with a class or id selectr. Ex. body > main.
